# Customer-Churn-Analysis
Analyzed telecom customer churn using Python to identify key factors affecting retention. Built KPIs and visualizations to derive business insights.


📌 Project Overview

Performed exploratory data analysis on telecom customer data using Python to identify churn patterns and key drivers affecting customer retention.

🎯 Objective

To understand why customers leave and help businesses reduce churn by identifying high-risk segments.

📊 Key KPIs

- Total Customers
- Churn Count
- Churn Rate (%)

🔍 Key Insights

- Month-to-month contract customers have the highest churn
- Higher monthly charges increase churn probability
- Customers without tech support or online security are more likely to churn
- Long-term contracts reduce churn

📊 Techniques Used

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Analysis
- Data Visualization

🛠 Tools Used

- Python
- Pandas
- Matplotlib / Seaborn

📁 Files

- customer_churn_analysis.ipynb
- customer-churn-analysis.pdf


📈 Business Impact

Helps telecom companies improve customer retention by targeting high-risk customers and optimizing service offerings.
